About Nutrition Recommendation System

This project is a Machine Learning based web application that recommends healthy meals according to the nutritional values entered by the user. It uses the K-Means Clustering algorithm to group similar meals and provides personalized food recommendations instantly.

Project Overview

Dataset

More than 2000 meals with calories, protein, carbohydrates, fat, fiber, sugar, sodium, cholesterol and serving size.

Data Preprocessing

Missing value handling, duplicate removal, feature selection and feature scaling using StandardScaler.

K-Means Clustering

Similar meals are grouped into clusters. User input is matched with the nearest cluster to generate recommendations.

Recommendation Engine

Top similar meals are recommended based on Euclidean Distance inside the predicted cluster.

Machine Learning Workflow

Load Dataset

Healthy Eating Dataset is loaded.

Preprocessing

Clean and normalize the nutritional features.

Train K-Means

Meals are divided into meaningful clusters.

Save Model

KMeans model and StandardScaler are saved using Joblib.

User Input

User enters nutritional requirements through the web form.

Meal Recommendation

Top matching healthy meals are displayed.
